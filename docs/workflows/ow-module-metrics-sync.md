# OW Module Metrics Sync

- Display Name: `OW Module Metrics Sync`
- Prompt Intent: Count current Workshop module metrics from src and sync the small set of docs/modules pages that still carry derived counts; support check-only validation and optional report output.

## Quick Start

```bash
tools/module-metrics-sync.ts
tools/module-metrics-sync.ts --check
tools/module-metrics-sync.ts --report
```
