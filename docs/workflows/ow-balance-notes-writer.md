# OW Balance Notes Writer

Use this workflow to produce Chinese player-facing balance notes from gameplay changes.

## Rules

1. Keep output in Chinese unless explicitly asked otherwise.
2. Split Player and AI blocks when values differ.
3. Put cooldown changes under concrete ability keys, not `General`.
4. Use time wording for cooldowns.
